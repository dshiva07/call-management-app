# Call Management App 📞

A comprehensive collection of web-based applications for managing calls, follow-ups, and having a good laugh!

## 📁 Applications Included

### 1. **Call Management System** 📞
**File:** `call-manager.html`

Track your daily calls, manage follow-ups, and maintain professional relationships.

#### Features:
- ✅ Add call records with contact details
- ✅ Track call status (Pending, Completed, No Answer, Rescheduled)
- ✅ Schedule follow-up appointments
- ✅ Filter and search calls
- ✅ View statistics and analytics
- ✅ Export data to CSV
- ✅ Local storage persistence
- ✅ Fully responsive design

#### How to Use:
1. Download `call-manager.html`
2. Open in any web browser
3. Add call records using the form
4. Track follow-ups and manage status
5. Export data whenever needed

---

### 2. **Random Joke Generator** 😂
**File:** `joke-generator.html`

Get random jokes instantly using an external API for a quick laugh!

#### Features:
- ✅ 4 joke categories (Random, Programming, Knock-Knock, General)
- ✅ External API integration (Official Joke API)
- ✅ Setup & punchline reveal button
- ✅ Add favorite jokes
- ✅ Copy jokes to clipboard
- ✅ Statistics tracking
- ✅ Local favorites storage
- ✅ Beautiful UI with animations

#### How to Use:
1. Download `joke-generator.html`
2. Open in any web browser
3. Select a category
4. Click "Get Joke 😂"
5. Enjoy and save favorites!

## 🚀 Quick Start

### Option 1: Use Directly in Browser
Simply download any HTML file and open it in your browser - no installation needed!

### Option 2: Host on Your Website
```html
<!-- Link in navigation -->
<a href="call-manager.html">Call Manager</a>
<a href="joke-generator.html">Joke Generator</a>

<!-- Or embed as iframe -->
<iframe src="call-manager.html" width="100%" height="900"></iframe>
```

### Option 3: Clone Repository
```bash
git clone https://github.com/dshiva07/call-management-app.git
```

## 🔧 Customization

### Change Colors:
Find the gradient in the CSS and modify:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Add More Categories:
Edit the category buttons in the HTML to add more joke types.

### Extend Call Manager:
Add more fields by duplicating the `form-group` sections.
